〇、这个系统能做什么（功能总览）
----------------------------------------------------------------
  买家比价：
    · 一次搜 5 大平台（Shopee、Lazada、淘宝、拼多多、1688）
    · 真实性评分（识别刷单/假货/异常低价）
    · 拍照 / 图片找同款（淘宝拍立淘识别，AI 备份）
  商家选品（进口爆款）：
    · 爆款潜力榜（需求 × 利润空间排行）
    · 市场概况（中位价、蓝海/红海、可点链接到商品）
    · 成本+利润定价 + 到岸真实利润计算器（扣运费/佣金/税）
    · 选品清单（收藏候选品，复查看需求涨跌）
    · 改良机会（AI 挖爆款差评 → 进改良版）
  防封号：
    · 搜中国平台 45 秒冷却、每日 50 次上限、登录提醒
    · 随机滚动、错峰、验证交给你本人过

----------------------------------------------------------------
  部件            住在哪里          换电脑要做什么
  --------------- ----------------- ------------------------
  网站（界面）     Vercel 云端       什么都不用做，开网址就能用

  结论：如果只是「用」，新电脑装个扩展就行，5 分钟搞定。
  你的选品清单、搜索历史都在云端，换电脑登录后原封不动都在。

三、情况 A：只是「使用」AAO（约 5 分钟）★90% 的情况
----------------------------------------------------------------
  第 1 步  拿到扩展文件夹（三个来源选一个）：
           a) 最简单：浏览器打开 github.com/Boon87/AAO
              → 点绿色「Code」按钮 → Download ZIP → 解压
              → 里面的 chrome-extension 文件夹就是扩展
           b) OneDrive 里的 aao-extension-LATEST 文件夹
           c) 备份压缩包 AAO-v1.1-merchant-backup.zip 里的
              chrome-extension 文件夹

  第 2 步  把这个文件夹复制到新电脑的本地路径：
              C:\Users\<你的用户名>\aao-extension
           ⚠️ 千万别放在 OneDrive 或桌面里！
           （OneDrive 会自己挪动文件夹，Chrome 就找不到扩展了）

  第 3 步  装进 Chrome：
           ① Chrome 地址栏输入： chrome://extensions 回车
           ② 右上角打开「开发者模式」开关
           ③ 点「加载已解压的扩展程序」
           ④ 选刚才那个 aao-extension 文件夹
           ⑤ 看到「AAO Price Helper」出现 = 成功 ✓

  第 4 步  打开网站并登录：
              aao-price-tool.vercel.app
           用员工邮箱 + 密码登录。

  第 5 步  在同一个 Chrome 里登录淘宝、拼多多、1688
           （已登录的搜索结果最全，也最不容易被平台拦截）

  完成！可以开始搜索、找爆款了。


四、情况 B：要「改代码 / 继续开发」（约 20 分钟，一次性）
----------------------------------------------------------------
  第 0 步  安装两个免费软件（官网下载，一路「下一步」）：
           · Git      → git-scm.com/download/win
           · Node.js  → nodejs.org  （选 LTS 版本）

  第 1 步  打开 PowerShell：
           开始菜单 → 输入 powershell → 回车

  第 2 步  下载代码（一行一行输入，每行按回车）：
              cd C:\Users\你的用户名
              git clone https://github.com/Boon87/AAO.git aao-price-tool
           第一次会弹出 GitHub 登录窗口 → 用 Boon87 账号登录一次。
           ⚠️ 一定放本地路径，别 clone 进 OneDrive！

  第 3 步  安装依赖（等几分钟）：
              cd aao-price-tool
              npm install

  第 4 步  复制密钥文件 .env.local：
           这个文件保存密码钥匙（目前约 10 个），「不在」GitHub 里，要手动带过去：
           · 从旧电脑复制 C:\Users\User\aao-price-tool\.env.local
           · 用 U 盘带到新电脑，放进新的 aao-price-tool 文件夹里
           · 里面的 ANTHROPIC_API_KEY / GEMINI_API_KEY 驱动 AI 功能
             （拍照识别、改良机会分析）——少了这两个 AI 功能会用不了
           ⚠️ 这个文件含密钥，不要发给别人、不要传到网上。

  第 5 步  测试跑起来：
              npm run dev
           浏览器打开 http://localhost:3000 能看到登录页 = 成功 ✓
           （按 Ctrl+C 可以停止）

  第 6 步  （推荐）装 Claude Code 让 AI 继续帮你开发：
           装好后，在 aao-price-tool 文件夹打开它，第一句话说：
           「请先读 SETUP-AND-MIGRATION.md 和 AGENTS.md 了解这个项目」

  ★ 日常循环（最重要，背下来）：
           改完代码 → 输入这三行（推上 GitHub + 自动部署网站）：
              git add -A
              git commit -m "写清楚改了什么"
              git push
           回到另一台电脑开工前 → 先输入：
              git pull
           口诀：「下班 push，上班 pull」
           两台电脑永远通过 GitHub 保持同步，永不打架。


五、三条铁律（血泪教训，2026-07-08 事故换来的）
----------------------------------------------------------------
  1. 代码（git 仓库）永远不放 OneDrive
     —— OneDrive 会弄坏 .git、回滚文件、生成冲突副本
  2. Chrome 扩展永远从本地路径加载（不放 OneDrive / 桌面）
     —— OneDrive 挪动文件夹会让 Chrome 禁用扩展
  3. 改完代码永远 git push
     —— 推上 GitHub 才算安全、网站才会更新


六、出问题速查
----------------------------------------------------------------
  问题：网站打不开 / 显示 504
  解法：Supabase 免费版闲置会「睡着」→ 登录 supabase.com
        → 找到项目（名字 floorpro-quotation）→ 点 Restore/恢复
        → 等 1-2 分钟刷新网站

  问题：搜索 0 结果 + 提示「需要 AAO 扩展」
  解法：chrome://extensions → 找 AAO Price Helper → 点刷新 ↻
        → 回网站刷新页面

  问题：淘宝 / 1688 / 拼多多没结果
  解法：先在 Chrome 登录这些平台账号再搜；
        如果弹出验证窗口 → 自己动手完成验证（拖滑块等）

  问题：git push 被拒绝（403 错误）
  解法：Windows「凭据管理器」→ Windows 凭据
        → 删除 git:https://github.com 那条 → 重新 push
        → 重新登录 Boon87
